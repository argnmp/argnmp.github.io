풀이

#include <iostream>
#include <algorithm>
#include <climits>
#include <vector>
#include <queue>
#include <map>
#include <unordered_map>
#include <set>
#include <unordered_set>
#include <bitset>
#include <cstring>
#include <cmath>
using namespace std;
using ll = long long;
int n, k;
vector<vector<int>> graph(501, vector<int>());
int ga_match[501] = {0,};
int gb_match[501] = {0,};
int visited[501] = {0,}; 

bool bi_match(int a){
    if(visited[a]==1) 
        return false;
    visited[a] = 1;
    for(int b: graph[a]){
        if(gb_match[b] == 0 || bi_match(gb_match[b])){
            ga_match[a] = b;
            gb_match[b] = a;
            return true;
        }
    } 
    return false;
}

void solve(){
    cin >> n >> k;
    int u, v;
    for(int i = 0; i<k; i++){
        cin >> u >> v;
        graph[u].push_back(v); 
    }
    int ans = 0;
    for(int i = 1; i<=n; i++){
        memset(visited, 0, sizeof(visited));
        if(bi_match(i))
            ans += 1;
    }
    cout << ans;
}
int main(){
    ios::sync_with_stdio(0);
    cin.tie(0);
    solve(); 
    return 0;
}

돌멩이의 좌표를 행과 열로 나누어서 생각해볼 수 있다.

다음과 같은 예시를 생각해보자. X는 돌멩이이고, _는 아무것도 없는 것을 의미한다.

X _ X
X X _
_ X _

1행을 달리면 다음과 같은 상태가 된다.

_ _ _
X X _
_ X _

이 상태에서 열 중심으로 생각해본다면, 1행을 달림으로써 1열은 달려야 할 필요가 있고 3열은 달릴 필요가 없게 된다.

이와 같이 행은 열에 영향을 미치고 열은 행에 영향을 미친다. 그래서 각 행과 열에 대하여, 영향을 미치게 되는 열과 행을 그래프로 그릴 수 있다. 그리고 이 그래프는 이분 그래프가 된다.

이분 그래프는 인접한 정점을 서로 다른 색으로 칠할 때 모든 정점을 두가지 색으로 칠할 수 있는 그래프를 의미한다. 즉, 모든 정점에 대하여 두 개의 그룹으로 나누었을 때, 그룹 내에는 간선이 존재하지 않는 상태를 의미한다.

행과 열을 각각의 두 그룹으로 생각하면, 모든 간선은 행에서 열로 향하기 때문에 이분 그래프에 해당하는 것이다.

 그래프에서는 행과 열을 각각의 정점으로 하며, 모든 간선은 하나의 돌멩이를 의미한다. 문제에서 돌멩이를 제거 하기 위한 달리기 한번은, 하나의 정점(행 or 열)을 선택해서 연결된 모든 간선을 제거하는 것과 같다. 구해야 하는 답은 최소 달리기의 횟수이다. 결국 최소 개의 정점을 선택해서 모든 간선을 제거해야 한다.

vertex cover은 그래프의 각 간선의 끝점 중 최소한 한개를 포함하는 정점의 집합을 의미한다. 따라서 vertex cover에 해당하는 정점에 연결된 간선들을 모두 제거하면, 그래프의 모든 간선이 제거된다. (모든 간선의 끝점 중 하나가 vertex cover에 들어 있기 때문). minimum vertex cover은 이러한 집합 원소의 최소 개수를 의미하고, 이는 문제에서 구해야 하는 값이며,  König's Theorem에 의하여 이분 매칭의 값으로 구할 수 있다.

행에 해당하는 정점들을 한 그룹으로, 열에 해당하는 정점들을 다른 그룹으로 해서 이분 매칭의 값을 계산하면 된다.

위의 풀이에서는, 항상 행에서 열로 향하는 directed edge로 그래프를 형성해서, 행과 열을 같은 정점 번호를 사용하도록 하였다.

느낀점

이분 그래프, 이분 매칭, minimum vertex cover에 대하여 새롭게 공부하였다.
