새로운 블로그

오랜만에 새로운 블로그를 제작했다. 그동안 여러 블로그들을 거쳐왔지만, 이번에는 직관적으로 정보를 전달하고 자원을 효율적으로 사용하는 데 중점을 두어서 블로그를 설계했다.

첫번째 블로그 2020.3 ~ 2022.2

처음 프로그래밍 공부를 시작하며, 사용했던 블로그는 티스토리이다. (navydoc.tistory.com) 어쩌다 보니 복무 기간과 겹치며 최장 기간 사용해온 블로그가 되었다. 가장 인기있는 글은 https://navydoc.tistory.com/31로, 리눅스에서 사용중인 포트를 종료하는 방법에 대해 작성한 글이다. linux 포트 종료하기 라고 구글에 검색하면 최상단에 위치한다. 그래도 방문자 수가 하루 평군 50명 정도는 되어서 구글 애드센스에 등록하여 수익을 창출하고 있지만, 수익은 매우 미미하다.

두번째 블로그 2022.5 ~ 2023.6

2022년에 전역을 하고 새로운 블로그를 제작했다. express.js와 react.js 기반의 ssr 방식의 블로그이다. azure 클라우드의 학생 계정에 지원되는 무료 서비스만 골라서 기본 제공되는 credit의 소모 없이 사용했었으나, 어느 순간부터 credit이 다 소진되어 버렸다. 이로 인해 사용료를 내지 않으면 더이상 이용할 수 없는 상태가 되었다. 그래서 인스턴스를 종료해놓고 db만 백업을 해놓은 상태이다.

세번째 블로그 2023.11 ~

두번째 블로그가 credit 이슈로 인해 더이상 무료로 사용할 수 없게 되자, 무료로 사용할 수 있는 블로그를 만들기 위한 계획을 세웠다. 불필요하게 구현되는 기능들은 걷어내고 직관적이고 명확한 방식으로 정보를 전달할 수 있도록 설계를 했다.

결국 github pages를 이용해서 현재의 블로그를 제작했다. 사실 이 블로그 자체를 제작한 것은 아니고, 정적 블로그 생성기를 제작했다. 로컬에서 글을 작성한 뒤(보통은 obsidian으로 작성한다), rust로 제작한 프로그램을 통해 이 블로그를 렌더링한다. 렌더링한 결과물을 git repository에 업로드하면 github pages를 통해 배포가 완료되는 방식을 사용한다.

한달 여의 기간동안 새롭게 제작한 블로그에는, 컴퓨터 프로그래밍과 관련된 주제의 글들을 올릴 예정이다.
