AMPS: 1세대, FDMA IS-41: 1세대, 2세대, 3세대 모두 사용 (코어망), 공용으로 사용한다. 바뀌지 않고 여전히 사용된다. GSM: 2세대, TDMA, SMS IS-95: 2세대, CDMA IMT-2000: 3세대, CDMA

AMPS: advanced mobile phone system

AMPS의 design goal
  • small coverage area (1~16miles)
  • frequency reuse planned in system design
  • 333+333 channels(-> 832 channels)
  • low power mobile transmitters(4 watts or less), low blocking (2%) during busy hour
AMPS의 characteristics
  • frequency: uplink(825mhz ~ 845mhz), downlink(870mhz ~ 890mhz), 이후 uplink 와 downlink 5mhz 씩 증가함. 각 기업마다 333개의 duplex channel, 21개의 control channel (각 셀마다 7개씩??), cluster에 7개의 cell, 각 cell마다 약 45개의 channel
  • 3khz의 voice channel을 30khz channel에 싣는다.
  • frequency modulation을 사용
  • controll channel과 voice channel이 나누어져있다.
general operation of AMPS
AMPS identification numbers
  1. ESN: electoric serial number, MS가 cellular system을 사용하기 위해 필요, 32bit binary number으로 uniquely identifies a cellular unit number established by the manufacture at the factory, 쉽게 바꿀 수 없음
  2. SID: system identification number, FCC의해 cellular system에 할당된 15bit binary number, MS는 SID를 전송해야 한다. cellular system은 할당된 SID만 전송해야한다.
  3. MIN: mobile identification number, MS의 10-digit directory telephone number
signaling on control channels
  • FOCCs: forward control channels
    • continuous data stream
    • MS에 system information을 전송한다. send pages, orders, voice channels assignments
    • TDM 방식을 이용해서 순서대로 세개의 stream을 전달한다. busy/idle stream, stream a, stream b
      • busy/idle stream은 RECC의 현재 상태를 나타낸다.
      • stream A는 MIN의 LSB가 0인 MS에 전달하는 message이고, stream B는 MIN의 LSB가 1인 MS에 전달하는 message이다.
    • format: dotting, word sync, 그리고 word A와 word B를 5번씩 반복
  • RECCs: reverse control channels
    • discontinuous, contention channel
    • slotted aloha packet radio channel로, control channel이 제한되어 있어서 먼저 차지하는 MS가 사용한다.
    • BS의 page에 응답한다. MS에서 call origination을 할 때 사용한다.
    • format: seizure precursor(dotting, word sync, coded DCC), word를 5번씩 반복
signaling on forward voice channel(FVC)
  • continuous supervisory audio tone (Beacons): 6khz에서 세개의 tone을 사용. transmitted by BS, received back at BS receiver, lack of tone used to squelch receiver, tone used to detect interference
  • discontinuous data stream: send orders to MS, send new voice channels assignments (handoff)
signaling on reverse voice channel(RVC)
  • continuous supervisory audio tone (Beacons): MS regenerates tone, lack of tone used to squelch receiver, tone used to detect interference
  • discontinuous data: confirms orders, MSs make 3-way calls(with dialed digits)
  • signaling tone (10khz): signal disconnect(1.5s), signal flash(0.5s)에 사용한다.
initialization procedures
  • MS에서 전원을 켜서 system A or B를 설정한다.
  • MS에서 control channel을 scan해서 가장 강한 채널에 연결한다.
  • BS에서 system parameter message를 보낸다
  • MS에서 operating parameter, SID를 업데이트하고 serving system을 인식하고, establish paging channels
MS originates the call
  • MS에서 origination message를 보낸다.
  • BS에서는 IS-41에 정보를 전달한다.
  • BS에서는 control message를 보낸다.
  • MS와 BS가 voice channel로 변경한다.
  • BS에서 FVC control message를 보낸다.
  • MS에서 RVC confirmation message(SAT)를 보낸다.
  • IS-41에서 BS에 answer를 한다.
  • conversation이 이어진다.
MS receives the call
  • BS에서 page control message를 보낸다.
  • MS에서 page response message를 보낸다.
  • (MS와 BS가 voice channel로 변경한다?)
  • BS에서 control message를 보낸다.
  • MS에서 confirmation message(SAT)를 보낸다.
  • conversation이 이어진다.
IS-41 entities
  • MSC: mobile switching center
  • ISDN: integrated services digital network (다른 망)
  • PSTN: public switched telephone network
  • anchor MSC: initial contact point, 단말기에서 처음 통화할 때 연결된다.
  • candidate MSC: handoff 과정에서 다음으로 서비스를 제공하게 될 MSC
  • homing MSC: 단말기를 가입할 때 등록하는 MSC, 단말기의 MIN number를 가지고 있는 MSC
  • serving MSC: 현재 서비스를 제공하는 MSC
  • target MSC: MS에 가장 좋은 signal quality를 제공할 수 있는 cell site를 가지고 있는 MSC
IS-41 and OSI
  • MAP: mobile application part
  • ASE: applications service element
  • ACSE: association control service element
  • ROSE; remote operation service element
  • TCAP: transaction capabilities application part
  • MTP: message transfer part
  • SCCP: signaling connection control part
IS-41 and amps
  • interworking of IS-41 and amps
  • registration with a new MSC
  • calling an idle MS in a new system
  • call with unconditional call forwarding
  • call with no answer
  • call a busy MS
  • handoff measurement request
  • recovery from failure at the HLR
