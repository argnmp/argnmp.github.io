AWS EC2에 password로 ssh 접속하기

Taehyeon Kim

2024-07-27

blog

개요

AWS EC2는 기본적으로 .pem 파일을 이용해서 ssh 접속을 하는 구조이다.

주로 사용하는 개발환경이 아닌 곳에서 접속하기 위해서 key 파일을 언제나 다운로드 해서 사용할 수 없기 때문에, password를 설정해서 접속할 수 있도록 설정해보려 한다.

sshd 설정 변경

먼저 EC2 인스턴스에 접속한다.

/etc/ssh 경로에 ssh 관련 설정들이 존재한다.

/etc/ssh/sshd_config 파일을 열어보면 아래와 같이 작성되어 있다.

...
Include /etc/ssh/sshd_config.d/*.conf
...

이 경로의 .conf 파일을 포함한다는 의미이다.

sshd_config.d 디렉터리에는 아래의 파일이 존재한다.

60-cloudimg-settings.conf

이 파일을 열어서 아래와 같이 yes로 수정을 해준다.

PasswordAuthentication yes

현재 나는 ubuntu image를 사용중이다. 따라서 아래와 같이 ssh service를 재시작 한다.

sudo systemctl restart ssh

ubuntu 계정 패스워드 설정

ubuntu image를 사용하고 있다면 ubuntu라는 이름의 계정으로 로그인 되어 있다.

비밀번호가 설정되어 있지 않기 때문에 아래의 명령어로 설정해준다.

sudo passwd ubuntu

접속

key 파일 없이 ssh로 접속을 하면 비밀번호를 입력하라고 한다. 그리고 비밀번호를 입력하면 접속할 수 있다.
