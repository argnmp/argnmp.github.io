서론

rust book의 example을 따라하던 중에 3u8 이라는 값을 보았다. u8은 rust의 data type 중에 하나이다. 3u8과 같이 바로 붙여서 쓰는 것은 본적이 없어서 구글링 하던 중 알게 되었다.

Suffix

rust by example 이라는 공식 예제에 따르면 literal을 다음과 같이 소개하고 있다.

https://doc.rust-lang.org/stable/rust-by-example/types/literals.html

Numeric literals can be type annotated by adding the type as a suffix. As an example, to specify that the literal 42 should have the type i32, write 42i32.

Numeric literal은 type을 suffix(뒤에 붙이는 것)을 통해서 type을 추가할 수 있다. 라고 쓰여 있다. 따라서 3u8은 type이 u8인 numeric literal 3을 의미하게 된다.
