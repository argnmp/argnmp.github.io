generic을 통한 trait bound

https://kimtahen.com/post/generic-%EC%82%AC%EC%9A%A9%EB%B2%95#hidtrait-bound

generic을 사용한 trait bound 방법이다.

trait bound는 함수에서 특정 trait을 가진 struct를 parameter로 받을 때 사용된다.

trait bound의 예시

trait Greeting {
    fn default(&self)->String{
        String::from("hello world")
    }
    fn special(&self)->String;
}
struct Human {
    mention: String,
}
struct Computer {
    mention: String,
}
impl Greeting for Human {
    fn special(&self)->String {
        self.mention.clone()
    }
}
impl Greeting for Computer {
    fn special(&self)->String {
        self.mention.clone()
    }
}

fn greet<T> (target: T) where T: Greeting{
    println!("{}",target.default()); 
    println!("{}",target.special()); 
}


fn main(){
    let p1 = Human {
        mention: String::from("I'm person 1") 
    };
    let p2 = Human {
        mention: String::from("I'm person 2") 
    };
    let c1 = Computer {
        mention: String::from("I'm computer 1") 
    };
    greet(p1);
    greet(p2);
    greet(c1);
}
  • Human, Computer struct는 Greeting trait을 가진다.
  • greet function은 Greeting trait을 가지고 있는 struct만을 target parameter로 허용한다.

multiple trait bound 예시

trait Greeting {
    fn default(&self)->String{
        String::from("hello world")
    }
    fn special(&self)->String;
}
struct Human {
    mention: String,
}
struct Computer {
    mention: String,
}
impl Greeting for Human {
    fn special(&self)->String {
        self.mention.clone()
    }
}
impl Greeting for Computer {
    fn special(&self)->String {
        self.mention.clone()
    }
}

fn greet<T> (target: T) where T: Greeting{
    println!("{}",target.default()); 
    println!("{}",target.special()); 
}


trait Calculate{
    fn calc(&self, x1: isize, x2: isize)-> isize;
}
impl Calculate for Human {
    fn calc(&self, x1: isize, x2: isize)-> isize {
        x1 + x2
    }
}
impl Calculate for Computer {
    fn calc(&self, x1: isize, x2: isize)-> isize {
        x1 + x2
    }
}


fn work<T> (target: T) where T: Greeting+Calculate{
    println!("{}",target.special());
    println!("{}",target.calc(1000,2000));
}


fn main(){
    let p1 = Human {
        mention: String::from("I'm person 1") 
    };
    let p2 = Human {
        mention: String::from("I'm person 2") 
    };
    let c1 = Computer {
        mention: String::from("I'm computer 1") 
    };

    work(p1);
    work(p2);
    work(c1);

}
  • work function은 Greeting 과 Calculate trait 모두를 가진 struct를 target으로 받는다.

여기서 중요한 점은 work function에서 Calculate trait만을 가진 struct를 trait으로 받는다고 정의한다면 Greeting trait에서 정의한 함수는 사용할 수 없다는 것이다. target이 Greeting과 Calculate 모두를 trait으로 가지고 있어도 말이다.

work 함수를 다음과 같이 수정하면 컴파일 하면 에러가 발생한다.

다음의 코드를 실행하면 에러가 발생하는 이유도 같다.

fn test<T> (t: &T) where T:Clone{
    println!("{}",t);
}

fn main(){
    let a = String::from("hello world");   
    test(&a);
}

error[E0277]: `T` doesn't implement `std::fmt::Display`
  --> src\main.rs:10:19
   |
10 |     println!("{}",t);
   |                   ^ `T` cannot be formatted with the default formatter
   |

test 함수는 Clone trait을 가지는 t를 인수로 받는다. 하지만 Clone만 bound 되고 Display가 bound되지 않았기 때문에 println!을 사용할 수 없다.

다음과 같이 코드를 바꾸어 주어야 정상적으로 컴파일 된다.

use std::fmt::Display;

fn test<T> (t: &T) where T:Clone+Display{
    println!("{}",t);
}

fn main(){
    let a = String::from("hello world");   
    test(&a);
}
