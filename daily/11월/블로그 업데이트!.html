새로운 블로그

오랜만에 블로그를 새롭게 만들었습니다. 그동안 여러개의 블로그를 거쳐왔지만, 이번에는 직관적으로 정보를 전달하고 자원을 효율적으로 사용하는 데 중점을 두어 블로그를 설계하였습니다.

첫번째 블로그 2020.3 ~ 2022.2

처음 프로그래밍 공부를 시작하며, 사용했던 블로그는 navydoc.tistory.com 입니다. 어쩌다 보니 군대 기간이 겹치게 되면서, 최장 기간 사용해온 블로그가 되었습니다. 가장 인기있는 글은 https://navydoc.tistory.com/31 리눅스의 사용중인 포트를 종료하는 방법에 대해 작성한 글로, linux 포트 종료하기 라고 구글에 검색하면 최상단에 위치합니다. 그래도 방문자 수가 하루 평군 50명 정도는 되어서, 구글 애드센스에 등록하여 수익을 창출하고 있습니다. 하루에 0.01 달러씩 말이죠..

두번째 블로그 2022.5 ~ 2023.6

2022년에 전역 후, 새로운 블로그를 제작했습니다. 백엔드는 express.js, 프론트엔드는 react.js 기반의 ssr 기능을 갖춘 블로그입니다. azure 클라우드의 학생 계정에 지원되는 무료 서비스만 골라서 기본 제공되는 credit의 소모 없이 사용했었으나, 어느 순간부터 credit이 다 소진되어 버렸고, 이로 인해 사용료를 내지 않으면 더이상 이용할 수 없는 상태가 되었습니다. 그래서 인스턴스를 종료해놓고 db만 백업을 해놓았습니다.

세번째 블로그 2023.11 ~

두번째 블로그가 credit 이슈로 인해 더이상 무료로 사용할 수 없게 되자, 저는 무료로 사용할 수 있는 블로그를 만들고자 결심했습니다. 불필요하게 구현되는 기능들은 걷어내고, 정보를 전달하는 것에 주안점을 둔 직관적이고 명확한 방식으로 설계를 하였습니다. 그리하여, github pages를 이용한 정적 페이지 호스팅을 이용하여, CSR과 SSR을 함께 사용하는 현재의 블로그를 제작하였습니다. 사실 이 블로그 자체를 제작했다기 보다는 블로그 생성기를 제작했다고 할 수 있습니다. 로컬에서 정적으로 글을 작성한 뒤, rust로 제작한 프로그램을 통해 이 블로그를 렌더링 합니다. 렌더링한 결과를 git repository에만 업로드하면 배포가 완료됩니다! 블로그 생성기 프로젝트와 관련해서는 추후 포스팅을 하겠습니다.

한달 여의 기간동안 새롭게 제작한 블로그에는, 컴퓨터 프로그래밍과 관련된 주제의 글들이 올라올 예정입니다. 감사합니다!
