max priority queue

priority_queue<int, vector<int>, less<int>> maxq;
priority_queue<int> maxq;
  • 두 개 모두 같은 의미이다.
  • less<int> 를 priority_queue에 push 할 때 push 하는 수가 더 작으면 swap을 한다고 이해하면 된다.

min priority queue

priority_queue<int, vector<int>, greater<int>> minq;
  • 마찬가지로 greater<int> 를 priority_queue에 push 할 때 push 하는 수가 더 크면 swap을 한다고 이해하면 된다.

priority queue with struct

struct compare_x {
  bool operator()(pos& a, pos& b){
    return a.x < b.x;
  }
};
priority_queue<pos, vector<pos>, compare_x> sorted_pos;
  • compare_x 와 같은 struct(class)를 하나 만들고, 이 struct에 연산자 오버로딩을 해준다. 그리고 이 struct(class)를 이용해서 priority queue를 정의한다.
  • bool operator()(pos& a, pos&b) 에서 첫번째 인자는 priority queue에 push하는 원소이고, 두번째 인자는 비교 대상이다.
  • 위에서 compare_x는 push하는 원소의 x 값이 더 작으면 swap을 한다. 따라서 top 에는 x값이 가장 큰 원소가 위치하게 된다. less와 greater를 이용한 것과 같은 방식이다.
