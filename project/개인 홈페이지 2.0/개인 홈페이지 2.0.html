개요

  • 개인 홈페이지로 사용하기 위한 웹사이트를 제작하였습니다. backend, frontend 모두 제작하였습니다.
    • backend는 express.js 기반, frontend는 react.js 기반입니다.
  • server side rendering framework를 사용하지 않고, React.js와 express.js를 이용해서 직접 구현한 ssr이 특징입니다.
    • express.js 에서 redux.js 를 통하여 React.js 의 state를 생성해주는 방식으로 구현하였습니다.
  • node.js 프로젝트를 생성하고 react.js와 express.js의 webpack 설정을 작성하는 등, 프로젝트 빌드를 위한 설정들을 처음부터 하나하나 직접해보았다는 점에서 의미있는 프로젝트였습니다.
  • 프로젝트 기간 이후에도 지속적인 수정을 거치며 업데이트 하였습니다.

프로젝트 기간

2022.02.22 ~ 2022.03.31

기술 스택

  • javascript node.js express.js react.js redux.js redux-saga.js webpack scss mongoDB

구현 사항

1. 회원가입, 로그인

  • 회원가입 및 로그인이 가능하고, 댓글 작성이 가능합니다. 권한을 부여받을 경우 글 작성이 가능합니다.

2. 글 작성, 마크다운 편집기

  • markdown으로 글 작성이 가능합니다.
  • 실시간으로 마크다운 렌더링 결과를 확인할 수 있는 에디터를 제작하였습니다.
  • 한번에 여러 장의 이미지를 업로드 할 수 있고, 이미지 링크가 생성되며 작성 중인 글에서 바로 확인이 가능합니다.
  • 사진을 업로드 했지만, 만약 글을 최종적으로 작성하지 않는다면 서버에서 자동으로 제거합니다.
  • 글의 제목을 uri로 사용합니다.

3. 댓글 작성, 수정, 삭제

  • 로그인 한 경우, 댓글 작성과 수정 그리고 삭제가 가능합니다. 관리자 권한을 가지고 있다면 다른 댓글을 삭제할 수 있습니다.

4. Table of Contents, markdown

  • 작성한 글은 markdown header에 따라, 자동으로 table of contents가 생성됩니다.

5. 다크 모드

  • 다크 모드를 지원합니다.
