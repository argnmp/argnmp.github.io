개요

입대 전 제작한 React.js + Google Firebase 기반의 개인 홈페이지 입니다.

프로젝트 링크

https://temp-bc726.web.app/

프로젝트 기간

2020.9.1 ~ 2020.9.20

기술 스택

  • javascript, node.js, react.js, redux.js, redux-saga.js, css, firebase
  • React.js 를 이용한 SPA 웹사이트
  • google firebase Cloud Firestore : Database
  • google firebase Authentication : user authentication
  • google firebase Hosting : React.js로 빌드한 웹사이트를 배포

React.js 를 frontend, google firebase를 backend로 사용한 프로젝트입니다.

주요 기능

1. 관리자 로그인

  • google firebase authentication 으로 로그인 하여 글 작성이 가능합니다.

2. 글 작성, 수정, 삭제

  • 마크다운 에디터를 사용한 글 작성이 가능합니다.
  • 글 작성, 수정, 삭제가 구현되었습니다.
  • 이미지 업로드 또한 가능합니다.

3. Command Line Interface

  • javascript와 css를 이용해서 terminal과 비슷하게 동작하는 페이지를 제작하였습니다.
  • 명령어를 입력하여 방명록 작성이 가능합니다.
