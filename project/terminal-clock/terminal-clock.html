개요

  • 스탑워치, 타이머, 현재시각 기능을 가진, terminal에서 실행 가능한 프로그램입니다.
  • ncurses library를 이용해서 숫자를 렌더링 합니다.
  • github에 build 방법과 사용법을 작성해두어, 각자의 환경에 맞게 빌드해서 사용할 수 있도록 하였습니다.
  • 복무 중 부대 내에서 진행한 프로젝트입니다. 스마트폰으로 개인 서버에 ssh 접속을 통해 코딩을 하였습니다.

프로젝트 링크

https://github.com/kimtahen/terminalclk-app-cpp

프로젝트 기간

2021.08.11 ~ 2021.08.23

기술 스택

  • c++ make

구현 사항

  • thread를 이용하여 비동기로 키보드 입력을 받아 프로그램을 조작할 수 있습니다.

  • refresh interval을 조정하여, 프로그램의 성능을 조정할 수 있습니다.

  • thread library와 ncurses library를 지원하는 어느 환경이든, 빌드해서 사용할 수 있습니다.
