arch linux에서 사운드가 출력되고 입력되는 방식을 설명한다.

Advanced Linux Sound Architecture (ALSA)

ALSA는 kernel에서 실행되는 sound mixer로, 사운드 카드를 직접 조작한다.

arch wiki에서는 alsa를 다음과 같이 설명한다. https://wiki.archlinux.org/title/Advanced_Linux_Sound_Architecture

The Advanced Linux Sound Architecture (ALSA) provides kernel driven sound card drivers.

ALSA는 kernel로 작동되는 사운드 카드 드라이버를 제공한다.

Besides the sound device drivers, ALSA also bundles a user space driven library for application developers. They can then use those ALSA drivers for high level API development. This enables direct (kernel) interaction with sound devices through ALSA libraries

sound device driver만 제공하는 것이 아니라 user space에서 실행 가능한 library를 제공한다. application developers는 이 library를 가지고 high level api를 개발할 수 있다. 또한 이 alsa가 제공하는 library로 sound device에 직접 접근이 가능하다.

alsa는 arch linux에서 이미 기본 kernel module에 포함되어 있기 때문에 다른 설정이 필요하지 않다.

Pulseaudio

https://wiki.archlinux.org/title/PulseAudio#Front-ends

PulseAudio is a general purpose sound server intended to run as a middleware between your applications and your hardware devices, either using ALSA or OSS.

PulseAudio는 ALSA 나 OSS를 이용하여 application과 hardware device 사이의 middleware 역할을 하는 sound server이다.

사운드 카드에 대한 하드웨어적인 접근은 ALSA kernel mode에서 담당하고, pulseaudio는 kernel과 user application 을 연결해주는 middleware의 역할을 한다.
