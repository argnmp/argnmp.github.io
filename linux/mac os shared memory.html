mac os shared memory

alias undefined
author Taehyeon Kim
created at 2023-12-06
tags

문제

m2 macbook에서 shared memory를 사용하는데, ftruncate가 제대로 작동하지 않는 문제가 있었다. 계속 Invalid argument 라는 에러가 발생했다. 구글링해도 문제가 안나오길래 혼자 열심히 테스트해보다가, 결국 원인을 찾아냈다.

해결

shm_open 후 제대로 shm_unlink를 하지 않아서 발생하는 문제였다. 제대로 shm_unlink가 되지 않은 shared memory file name을 다시 사용하니 문제가 발생했다.

linux에서는 이런 문제가 발생하지 않았는데, 아마 다른 환경 때문에 이런 문제가 발생한 것 같다.

linux에서는 /dev/shm 하위에 tmpfs file system을 mount하여 shared memory file을 생성한다. 하지만 macOS는 직접적으로 확인이 불가능하다.
