vim, nvim 에서 텍스트 복사가 되지 않을 때

alias undefined
author Taehyeon Kim
created at 2022-05-06
tags

서론

arch linux를 새로 설치하고 gnome desktop을 사용중인데 neovim에서 "+y 로 사용하던 텍스트 복사가 작동하지 않았다. 그래서 구글링하여 찾아보던 중 해결책을 찾게 되었다.

해결 방법

vim 이나 neovim 에서 아래의 명령어를 입력한다.

:checkhealth provider

위와 같은 결과가 출력 된다.

가장 상단에 No clipboard tool found 라고 되어있다. 따라서 xclip 이라는 프로그램을 설치해주어야 한다.

나는 arch linux를 사용 중 이기 때문에 yay를 사용한다. 자신의 linux package manager 에 따른 방법을 사용하면 된다

yay -S xclip

vim / neovim 에서 다음의 명령어를 입력해주면

:checkhealth provider

OK 라고 뜬다.

이후 다시 "+y 를 입력하여 텍스트를 복사해보면 정상적으로 작동함을 확인할 수 있다.
